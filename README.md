# Builders Invoice

Contractor-focused quoting, invoicing, payments, client management, expenses, and profit reporting.

## Local development

1. Copy `.env.example` to `.env.local` and provide development credentials.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:5173` or `http://localhost:5173/app.html`.

Useful commands:

- `npm run dev` — Vite development server
- `npm test` — API security and domain unit tests
- `npm run lint` — JavaScript linting
- `npm run build` — production multi-page build
- `npm run check` — lint, tests, and production build
- `npm run audit` — production dependency audit
